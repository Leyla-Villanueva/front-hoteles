# Gestion de Habitaciones y limpieza

### Aplicación web progresiva (PWA) para la gestión interna de un hotel, utilizada por camareras y personal de recepción.

## Descripción General

### Esta PWA permite visualizar el estado de las habitaciones, gestionar tareas de limpieza y apoyar las funciones de recepción. Funciona offline, es instalable y está organizada bajo una arquitectura modular basada en JavaScript.
